PhongLighting
=============

Phong lighting algorithm demo

### Case 1: Phong simple ###

1. Select a file
2. Select "简单" (simple)
3. Set the parameters
4. Press "面消隐" (Hidden-Surface Removal)

And it display like this:

![Phong Simple](pic/1.jpg "Phong Simple Picture")
